Semantics

The new semantic tags allows us to build simpler layouts and clearer html code.

You can use the tags header, footer, section, article, nav and aside in your layouts to define different page areas:

Layout:
This is the header
Main navigation area
Side menu
An article
Another article
Code: 
<header>This is the header</header>
<nav>Main navigation area</nav>
<aside>Side menu</aside>
<section>
    <article>An article</article>
    <article>Another article</article>
</section>
